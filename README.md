# Green Gauge
Our project, Green Gauge, uses MapBox, DeepForest, and AirVisual to examine tree canopies and levels of air quality in different urban areas. This data can help communities and policymakers gain more insight into their local environment.

### Inspiration
Living in an urban environment like Boston, we recognize the crucial role trees play in improving air quality, reducing carbon emissions, and mitigating rising temperatures. We created Green Gauge to raise awareness and provide actionable insights to both policymakers and the community, helping enhance Boston’s environmental quality.

### What It Does
Green Gauge allows users to input an address anywhere in the world. It retrieves a satellite image of the location, processes it using a pre-trained AI model, and generates an analysis. The results are then sent to an LLM, which provides actionable insights. The tool evaluates tree coverage percentage, number of trees, weather conditions, and air quality, offering valuable data to drive environmental improvements.

### How We Built It
We developed Green Gauge using React + Vite for the front end. To gather and analyze environmental data, we integrated several APIs, including OpenAI Vision, AirVisual API, and Mapbox. These APIs enable us to retrieve satellite images and extract relevant environmental insights.

### Challenges We Faced
One of our biggest challenges was analyzing satellite images effectively. We struggled to find a pre-trained model that provided sufficiently accurate results. Training a CNN model from scratch was infeasible given time constraints, so we relied on an existing vision model capable of detecting trees rather than multiple land-use classes like roads or rooftops.

### Accomplishments We're Proud Of
We are proud of the accurate environmental assessments generated by our pre-trained CNN model and the insightful recommendations provided by OpenAI Vision. Despite challenges, our tool successfully extracts and interprets key environmental data.

### What We Learned
We gained experience using Mapbox-GL's API and converting its images into a format compatible with DeepForest and OpenAI Vision. This knowledge expanded our ability to work with geospatial and AI-driven environmental analysis tools.

### What's Next for Green Gauge
We plan to enhance Green Gauge by incorporating higher-resolution satellite imagery for more precise analysis. Additionally, we aim to develop our own AI model to better evaluate green space and explore advanced techniques such as transformers, deep learning, and SVMs. Lastly, we intend to integrate MongoDB to store snapshots of key regions, allowing users to track changes in green space over time.
